# SigaUTS

SigaUTS es un Sistema Integrado de Gestión Académica desarrollado para las Unidades Tecnológicas de Santander. El proyecto permite administrar usuarios, roles, estudiantes, docentes, materias, grupos y matrículas estudiantiles mediante una plataforma web construida con Spring Boot.

## Descripción general

El sistema busca centralizar procesos académicos básicos como el registro de usuarios, la gestión de estudiantes y docentes, la administración de materias y grupos, y la consulta de información académica según el rol del usuario.

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA / Hibernate
- Thymeleaf
- Bootstrap
- MySQL
- Maven
